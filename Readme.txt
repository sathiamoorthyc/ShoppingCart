+###########################################
+# Shopping Cart Application               #
+###########################################
+
+This application is created for the shop which just sells apples and oranges at .60p & .25p respectively.
+
+It takes the list of items and calculates the total cost of the items.
+It also calculates for any offer. 
For ex: The shop decides to introduce two new offers

--> buy one, get one free on Apples

--> 3 for the price of 2 on Oranges

How To Run:
____________________

Run the 'CheckoutService' as JUnit Test.
